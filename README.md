# SmartX Rewards, Redesign (proposal + mockup)

Design and front-end for the SmartX Rewards redesign.

- `index.html` - the written proposal / spec (white theme). Open in a browser; it embeds the live mockup.
- `rewards.html` - the live UI mockup (dark, app-faithful): Referral / Points / Leaderboard tabs.
- `smartx-logo.svg`, `smartx-logo-white.svg` - logo assets.

Static and self-contained: no build step, no dependencies. Open `index.html` or `rewards.html` directly in any browser.

Fonts load from Google Fonts (Inter, Space Grotesk, JetBrains Mono). All numbers are illustrative.

The proposal's optional background-music asset (`bg.mp3`) is not included in this repo.
